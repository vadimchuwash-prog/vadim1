ПЕРЕД ПЕРВЫМ ЗАПУСКОМ:

ПЕРВЫЙ ЗАПУСК:
   При первом запуске бот попросит ввести:
   - API Key (BingX)
   - Secret Key (BingX)
   - Telegram Bot Token
